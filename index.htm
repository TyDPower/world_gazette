About the App

App Information

World Gazetteer displays geographical information based on worldwide or specific country searches. All the information displayed has been pulled from various scources via API calls. To see a list of the APIs used please see the credits tab.

There are three main functions avalible currently:

1: Country Specific Searches. This will search for the selected country and display a mini info bubble. If that bubble is closed and the location pin is clicked on a window with more information will be displayed. These country searches can also be overlayed with additional on-map locations, which when clicked will display information about that location. Clicking the search button will clear all natural event markers but not map routes.

2: Worldwide Searches. This will display the locations of the selected Natural Events that have occurred within that last 365 days. If the markers are clicked, a window will display information on the event at the location. Clicking the search button will clear all markers and map routes.

3: Map and Route Change. This section will change the map to the newly selected map with additional route options avalible to overlay on the map. Clicking the search button will clear all natural event markers but not map routes.
